# Palo Alto Cortex Cloud Workshops and Labs

Welcome to the Palo Alto Cortex Cloud Workshops and Labs! Cortex Cloud is Palo Alto Networks' comprehensive Cloud-Native Application Protection Platform (CNAPP) that delivers real-time security from code to cloud to SOC. It unifies application security, cloud posture management, cloud runtime security and SecOps into a single converged platform across your hybrid, multi-cloud environments (AWS, Azure, GCP, Kubernetes).

![readme-cortex-cloud](./images/readme-cortex-cloud.png)

## Key Capabilities

| Capability | Description |
|---|---|
| **Cloud Posture Security** | Misconfiguration detection, compliance monitoring, SmartGrouping, SmartScore risk prioritization, attack path analysis |
| **Application Security** | IaC scanning, SCA, secrets scanning, CI/CD integration, software supply chain security |
| **Cloud Runtime Security** | Cortex XDR agent-based protection, cloud detection and response (CDR), runtime defense |
| **Cloud APIs** | Cortex Cloud Platform APIs for automation and integration |
