# AWS Lab Templates

This folder contains the CloudFormation templates and supporting scripts used to deploy the AWS lab environment.

## Files

| File | Description |
|---|---|
| `aws-lab-template.yml` | Main CloudFormation template for the lab environment |
| `ec2-userdata-linux.sh` | User data script for Amazon Linux 2 EC2 instance |
| `ec2-userdata-windows.ps1` | User data script for Windows Server EC2 instance |
| `dvwa-deployment.yaml` | Kubernetes deployment for DVWA sample app |
